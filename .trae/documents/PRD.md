## 1. 产品概述

分子性质计算平台是一个面向化学研究者和材料科学家的在线计算工具，支持上传分子结构文件，自动完成多维度性质计算（能量、偶极矩、HOMO-LUMO能级），并根据分子量和原子组成智能选择计算精度。平台提供实时进度追踪、结果自动分类标记、历史任务管理、详细报表生成与导出、以及统计看板功能。

- 目标用户：化学/材料科学领域研究人员、高校师生、药企研发人员
- 核心价值：将复杂量子化学计算流程化、自动化，降低使用门槛，提升科研效率

## 2. 核心功能

### 2.1 用户角色

| 角色 | 注册方式 | 核心权限 |
|------|----------|----------|
| 普通用户 | 邮箱注册 | 上传分子、查看计算结果、导出报表 |
| 研究员用户 | 邮箱注册 + 机构认证 | 高精度计算、批量任务、历史数据对比 |

### 2.2 功能模块

1. **分子上传页**：文件上传、格式校验、原子组成识别、计算精度自动选择
2. **计算中心页**：实时进度展示、多性质并行计算状态、消息推送通知
3. **历史任务页**：任务列表、状态筛选排序、任务删除、快速查看结果
4. **结果详情页**：分子结构图、性质数值、分类标签、报表导出（Excel/图片）
5. **统计看板页**：每日计算量趋势、平均耗时统计、分子类型占比分布

| 页面名称 | 模块名称 | 功能描述 |
|----------|----------|----------|

## 3. 核心流程

用户上传分子结构文件 → 系统校验格式并解析原子组成 → 根据分子量和原子数自动选择计算精度 → 启动多性质并行计算 → 实时更新进度 → 计算完成推送通知 → 自动分类标记（半导体候选/高风险）→ 用户查看结果 → 生成报表导出

```mermaid
flowchart TD
    "A[用户上传分子文件]" --> "B[格式校验]"
    "B" -->|"校验通过"| "C[解析原子组成与分子量]"
    "B" -->|"校验失败"| "D[提示格式错误]"
    "C" --> "E{分子量>500Da 或 含重金属?}"
    "E" -->|"是"| "F[启用高精度模式]"
    "E" -->|"否"| "G[启用标准精度模式]"
    "F" --> "H[启动多性质并行计算]"
    "G" --> "H"
    "H" --> "I[实时更新进度]"
    "I" --> "J{计算完成?}"
    "J" -->|"是"| "K[推送通知 + 自动分类标记]"
    "J" -->|"否"| "I"
    "K" --> "L[用户查看结果/导出报表]"
```

## 4. 用户界面设计

### 4.1 设计风格

- 主色调：深邃科技蓝（#0A1628）作为背景，搭配电光青（#00F0FF）和量子紫（#8B5CF6）作为强调色
- 辅助色：警告红（#EF4444）用于高风险标记，琥珀黄（#F59E0B）用于半导体候选标记，翠绿（#10B981）用于已完成状态
- 按钮风格：圆角矩形（8px），主要按钮带渐变发光效果，次要按钮半透明边框
- 字体：标题使用 Space Grotesk（科技感），正文使用 Noto Sans SC（中文友好）
- 布局风格：左侧导航栏 + 右侧内容区，卡片式模块布局，毛玻璃效果面板
- 图标风格：线性描边图标，2px 粗细，与电光青强调色统一
- 动画：数据加载脉冲动画、进度条流光效果、卡片入场渐现、hover 发光反馈

### 4.3 响应式设计

- 桌面优先设计，主要适配 1280px+ 宽度
- 平板（768px-1279px）：侧边导航收起为图标模式，卡片单列排列
- 移动端（<768px）：底部标签导航，卡片堆叠，表格改为卡片列表

### 4.4 3D场景指导

- 分子结构3D渲染使用 Three.js，球棍模型展示
- 环境光 + 点光源组合，深色背景衬托发光原子
- 原子球体使用金属质感着色器，键使用圆柱体
- 支持鼠标拖拽旋转、滚轮缩放、自动慢速旋转
- 原子悬停高亮 + 信息提示
